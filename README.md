# Denis & Sahiba

Digital wedding invitation for 28 August 2026.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Before launch

- Connect the RSVP form to the wedding mailbox and response sheet.
- Create and connect `kontakt@denis-sahiba.com`.
- Confirm the final Google Maps destination on a phone before launch.
- Add the couple's photographs when available.
